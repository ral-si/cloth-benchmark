The folder contains all the documents needed for the benchmark on dual-arm cloth manipulation. In detail:

	DACM_benchmark_video.m4v -> summary video of the benchmark

    Baselines contains the results of the baseline solutions
        Task1/Baseline_Task1_B-DACM-S-0.1_comments_to_submit.pdf -> comments to the baseline solution of Task 1
        Task1/Baseline_Task1_B-DACM-S-0.1_scoring_sheet.pdf -> filled scoring sheet of the baseline solution of Task 1
        Task2/Baseline_Task2_B-DACM-F-0.1_comments_to_submit.pdf -> comments to the baseline solution of Task 2
        Task2/Baseline_Task2_B-DACM-F-0.1_scoring_sheet.pdf -> filled scoring sheet of the baseline solution of Task 2
        Task3/Baseline_Task3_B-DACM-S-0.1_comments_to_submit.pdf -> comments to the baseline solution of Task 3
        Task3/Baseline_Task3_B-DACM-D-0.1_scoring_sheet_t-shirt_1.pdf -> filled scoring sheet of the baseline solution of Task 3 obtained with the first T-shirt sample
        Task3/Baseline_Task3_B-DACM-D-0.1_scoring_sheet_t-shirt_2.pdf-> filled scoring sheet of the baseline solution of Task 3 obtained with the second T-shirt sample

    Protocols_and_benchmarks contains benchmark and protocol documents
        Task1/Benchmark_Task1_B-DACM-S-0.1.pdf -> benchmark document for tablecloth spreading task
        Task1/Protocol_Task1_P-DACM-S-0.1.pdf -> protocol document for tablecloth spreading task
        Task2/Benchmark_Task2_B-DACM-F-0.1.pdf -> benchmark document for folding task
        Task2/Protocol_Task2_P-DACM-F-0.1.pdf -> protocol document for folding task
        Task3/Benchmark_Task3_B-DACM-D-0.1.pdf -> benchmark document for dressing task
        Task3/Protocol_Task3_P-DACM-D-0.1.pdf -> protocol document for dressing task
		
    Scoring_sheets contains the files for reporting scores (.xlsx, .ods and .pdf extensions are provided)
        Task1/Scoring_sheet_Task1_B-DACM-S-0.1.[ods,xlsx,pdf] -> scoring sheet for tablecloth spreading task
        Task2/Scoring_sheet_Task2_B-DACM-F-0.1.[ods,xlsx,pdf] -> scoring sheet for folding task
        Task3/Scoring_sheet_Task3_B-DACM-D-0.1.[ods,xlsx,pdf] -> scoring sheet for dressing task

    Supplementary_material contains the additional material for the benchmark
        generate_head_stls.py -> python script for generating the .stl models of the heads for dressing task
        P-DACM-D-0.1_t-shirt_template.png -> picture of the T-shirt template for dressing task
